# Wave on Sierpinski

Simple python project simulating solution to wave equation on Sierpinski gasket according to my understanding of calculus on fractals.

---

## How to Run

1. Install dependencies:
    ```bash
    pip install numpy matplotlib scipy
    ```

2. Run the GUI:
    ```bash
    python main.py
    ```
